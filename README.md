# GifTastic

## Overview
Calls the GIPHY API to make a dynamic web page that populates with gifs of the user's choice.

![](https://res.cloudinary.com/damplk/image/upload/v1580628276/portal/hw_giphy02_hpuvgr.png)

## Technology
* HTML5
* CSS3
* JavaScript
* jQuery
* Meyer CSS Reset
